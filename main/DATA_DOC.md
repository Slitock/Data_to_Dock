**Справочник по данным: sku_data**

### Название и Описание

Справочник sku_data содержит сведения о товарах в нашем бизнесе. Каждая строка в таблице соответствует одному товару, и информация о нем включает в себя идентификатор SKU, количество оставшихся единиц, базовую цену, категорию, дату последнего обновления, зону хранения и флаг активности.

### Расположение данных

Данные хранятся в формате Parquet и располагаются на файловой системе с именем `bd.csv`.

### Структура данных

| Название | Тип | Not Null | Примеры | Описание |
| --- | --- | --- | --- | --- |
| sku_id | str | True | A-105, B-202, C-303 | Уникальный идентификатор товара |
| qty_rem | float64 | False | 140.0, 12.0, 0.0 | Количество оставшихся единиц |
| base_price | float64 | True | 1250.5, 45500.0, 15.25 | Базовая цена товара |
| category | str | True | electronics, machinery, consumables | Категория товара |
| last_upd | str | False | 2026-02-08, 2026-02-07, 2026-01-15 | Дата последнего обновления |
| storage_zone | str | False | Zone-A, Zone-B, Zone-C | Зона хранения |
| is_active | object | False | True, False | Флаг активности товара |

### Описание логики

Таблица sku_data используется для отслеживания информации о товарах в нашем бизнесе. Каждый ряд таблицы соответствует одному товару, и информация о нем включает в себя идентификатор SKU, количество оставшихся единиц, базовую цену, категорию, дату последнего обновления, зону хранения и флаг активности.

### FAQ и Примеры

Пример SQL-запроса для работы с таблицей sku_data:
```sql
SELECT * FROM sku_data WHERE category = 'electronics' AND is_active = True;
```
Здесь мы выбираем все строки из таблицы sku_data, где категория товара равна 'electronics' и флаг активности равен True.
