**Название и Описание**
Справочник товаров (Product Catalog)

Справочник товаров предназначен для хранения информации о различных продуктах компании. Каждая запись в таблице содержит уникальный идентификатор SKU, количество оставшихся единиц на складе, базовую цену, категорию и информацию о последнем обновлении.

**Расположение данных**
Данные хранятся в файле Parquet с именем `bd.csv` на пути `/data/warehouse`.

**Структура данных**

| Название | Тип | Not Null | Примеры | Описание |
| --- | --- | --- | --- | --- |
| sku_id | str | true | A-105, B-202, C-303 | Уникальный идентификатор товара |
| qty_rem | float64 | false | 140.0, 12.0, 0.0 | Количество оставшихся единиц на складе |
| base_price | float64 | true | 1250.5, 45500.0, 15.25 | Базовая цена товара |
| category | str | true | electronics, machinery, consumables | Категория товара |
| last_upd | str | false | 2026-02-08, 2026-02-07, 2026-01-15 | Дата последнего обновления |
| storage_zone | str | false | Zone-A, Zone-B, Zone-C | Зона хранения товара |
| is_active | object | false | true, false | Флаг активности товара |

**Описание логики**
В таблице хранится информация о различных продуктах компании. Каждая запись содержит уникальный идентификатор SKU, количество оставшихся единиц на складе, базовую цену и категорию товара. Столбец `last_upd` указывает дату последнего обновления информации о товаре. Столбец `storage_zone` указывает зону хранения товара.

**FAQ и Примеры**
Вот пример SQL-запроса для запроса информации о товарах, которые имеют более 100 единиц на складе:
```sql
SELECT * FROM product_catalog WHERE qty_rem > 100;
```
В этом примере мы используем оператор `>` для сравнения количества оставшихся единиц на складе с 100. Результатом запроса будет таблица, содержащая информацию о товарах, которые имеют более 100 единиц на складе.

Также можно использовать фильтр по категории:
```sql
SELECT * FROM product_catalog WHERE category = 'electronics';
```
В этом примере мы используем оператор `=` для сравнения категории товара с строкой `'electronics'`. Результатом запроса будет таблица, содержащая информацию о электронных продуктах.
